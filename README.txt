Notebooks de la compétition Kaggle : TensorFlow – Help Protect the Great Barrier Reef

Description des Notebooks :

gbr-exploration-v1
Il s'agit d'une phase d'exploration des données et quelques exemples de visualisations


gbr-yolox-train-infer
Notebook effectuant tout le setup tensorflow, la préparation des données et le 
téléchargement du modèle Yolox. Enfin, le modèle est entrainé avec les données de 
la compétition. 
output : le modèle entrainé


gbr-yolox-infer
Notebook qui charge le modèle entrainé précédement et effectue des prédictions sur le testset
output : fichier csv de soumissions


gbr-yolov5-train
Notebook effectuant tout le setup tensorflow, la préparation des données et le 
téléchargement du modèle Yolov5. Enfin, le modèle est entrainé avec les données de 
la compétition. 
output : le modèle entrainé


gbr-yolov5-infer
Notebook qui charge le modèle entrainé précédement et effectue des prédictions sur le testset
output : fichier csv de soumissions
